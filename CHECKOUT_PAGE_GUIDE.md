## 💳 Payment Options Breakdown

### 1. Cards (3 options)
```
┌──────────┐  ┌──────────┐  ┌──────────┐
│    💳    │  │    💳    │  │    💳    │
│   Visa   │  │Mastercard│  │  RuPay   │
└──────────┘  └──────────┘  └──────────┘
```
- Visa: International credit/debit cards
- Mastercard: International credit/debit cards
- RuPay: Indian domestic cards

### 2. Digital Wallets (3 options)
```
┌──────────┐  ┌──────────┐  ┌──────────┐
│    🍎    │  │    🔍    │  │    💰    │
│Apple Pay │  │Google Pay│  │  PayPal  │
└──────────┘  └──────────┘  └──────────┘
```
- Apple Pay: For iOS users
- Google Pay: For Android users
- PayPal: International wallet

### 3. UPI (3 options)
```
┌──────────┐  ┌──────────┐  ┌──────────┐
│    📱    │  │    🔍    │  │    👛    │
│ PhonePe  │  │   GPay   │  │  Paytm   │
└──────────┘  └──────────┘  └──────────┘
```
- PhonePe: Popular Indian UPI app
- GPay: Google's UPI solution
- Paytm: Wallet + UPI

### 4. Bank Transfer (4 options)
```
┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐
│    🔄    │  │    🏦    │  │    🏢    │  │    💶    │

└──────────┘  └──────────┘  └──────────┘  └──────────┘
```
- IMPS: Immediate Payment Service (India)
- NEFT: National Electronic Funds Transfer (India)
- ACH: Automated Clearing House (USA)
- SEPA: Single Euro Payments Area (Europe)

### 5. Buy Now Pay Later (2 options)

- ZestMoney: Indian BNPL service

## 🎯 User Interaction Flow

```
1. User lands on checkout page
   ↓
2. Reviews order summary (left side)
   - Doctor details
   - Appointment date/time
   - Patient name
   - Exact consultation fee
   ↓
3. Scrolls through payment categories (right side)
   - Cards
   - Digital Wallets
   - UPI
   - Bank Transfer
   - BNPL
   ↓
4. Hovers over payment options
   - Cards highlight in blue
   - Slight lift animation
   ↓
5. Clicks to select payment method
   - Card gets blue border
   - Background turns light blue
   - Radio button selected
   ↓
6. Checks "Accept Terms" checkbox
   ↓
7. Clicks "Pay ₹800 Securely" button
   ↓
8. Redirected to processing page
```
